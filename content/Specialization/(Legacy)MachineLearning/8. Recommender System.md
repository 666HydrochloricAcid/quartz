# 8. 0 Introduction
---
Recommender system은 유저들과 아이템의 정보를 토대로 긍정적인 반응을 보임직한 아이템들을 추천하는 알고리즘이다. 유저에게 영화를 추천해주는 모델을 생각해보자. 이때, 유저의 정보는 영화의 평점이나, 영화를 얼마나 끝까지 시청하였는가 와 같은 것들이다. 아이템의 정보는 영화가 어떤 장르인지, 길이는 얼마나 되는지 등이다.

영화 추천 시스템으로 예를 들어 설명하겠다. 사용자는 시청한 영화에 한하여 평점을 매기고, 각 영화는 얼마나 특정 장르의 성격을 띄는 지에 대한 데이터가 있다고 해보자. 다음은 노테이션에 대한 설명이다:

$n_u$ : number of users
$n_i$ : number of movies
$r(i,j)$ : 1 if user $j$ rated movie $i$ or 0
$y^{(i,j)}$ : rating given by user $j$ to movie $i$
$n$ : number of features
$w^{(j)},b^{(j)}$ : parameters for user $j$
$x^{(i)}$ : feature vector for movie $i$
$m^{(j)}$ : number of movies rated by user $j$

기본적으로 각 영화의 특징(장르, 배우, 감독, 연도 등)들이 충분히 feature로 반영되고 있다면, user의 평점은 다음과 같은 linear regression model로 예측할 수 있다.
