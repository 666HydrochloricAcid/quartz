# 8. 0 Introduction
---
Recommender system은 유저들과 아이템의 정보를 토대로 긍정적인 반응을 보임직한 아이템들을 추천하는 알고리즘이다. 유저에게 영화를 추천해주는 모델을 생각해보자. 이때, 유저의 정보는 영화의 평점이나, 영화를 얼마나 끝까지 시청하였는가 와 같은 것들이다. 아이템의 정보는 영화가 어떤 장르인지, 길이는 얼마나 되는지 등이다.
