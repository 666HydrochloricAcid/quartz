# 2. 0 Introduction to Binary Classification
---
A logistic regression model is used for binary classification problems, where the target value can take wither 0 or 1. Before that, let’s take a look at why linear regression model is not suitable for binary classification problems.

Suppose that we built a linear regression model that fits to the given data. To constrain the predictions to binary classes (0 or 1), we apply a decision boundary at the vertical line where the model output equals 0.5. The classification result based on this threshold is as below.

![feature2-1][ML2-1.png]
1. The prediction of linear regression can be less than 0 or greater than 1.
2. With outliers in the data, MSE tends to significantly distort the model’s original slope to minimize the vertical errors.
3. As a result, the decision boundary may change considerably, which could lead to misclassifications.
4. Additionally, the probabilistic interpretation of predictions can be difiicult, as it becomes difficult to assess how certain the model is about its classification, or in other words, evaluate the model's confidence.

To resolve this problem, let's introduce the logistic function, also known as the **sigmoid**.

### Sigmoid Function
Sigmoid function is a mathematical function whose graph has a characteristic S-shaped.
Sigmoid function $g:\mathbb{r}\rightarrow(0,1)$ is defined as:

$$g(z)=\cfrac{1}{1+e^{-z}}$$

And the graph is shown as follow:

![feature2-2][ML2-2.png]

Let's see how this function is applied in classification.
### Model
We will refer to the linear combination $\vec{w}\cdot\vec{x}+b$ that we used in the previous model as the logit. By combining the logit and the sigmoid function, we will construct the model as follow:

$$g(z)=g(\vec{w}\cdot\vec{x}+b)=\cfrac{1}{1+e^{-(\vec{w}\cdot\vec{x}+b)}}=f_{\vec{w},b}(\vec{x})$$

# 2. 1 Cost Function
---
MSE has shown to be quite unsuitable for binary classification problems. Therefore, we will derive a new cost function called **Binary Cross-Entropy** function or **BCE Loss** by replacing the squared error term with logistic error or **cross-entropy error** defined as follow:


$$
CEE = \begin{cases}
 &-\log\left(f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right) &,y^{(i)}=1 \\
 &-\log\left(1-f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right) &,y^{(i)}=0
\end{cases}
$$

![feature2-3][ML2-3.png]

CEE, as shown in the graphs, converges to 0 or diverges depending on how confidently the model predicts the correct target value. With simplified CEE, we can derive the CEE Loss, $J$ as follow:

$$\begin{aligned}&CEE=-y^{(i)}\log\left(f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right)-\left(1-y^{(i)}\right)\log\left(1-f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right)\\
& J(\vec{w},b)=-\cfrac{1}{m}\sum^m_{i=1}\left\{-y^{(i)}\log\left(f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right)-\left(1-y^{(i)}\right)\log\left(1-f_{\vec{w},b}\left(\vec{x}^{(i)}\right)\right)\right\}
\end{aligned}$$

# 2. 2 Gradient Descent
---
Let us derive gradient descent, with careful consideration of the changes in partial derivatives.

By chain rule,
$$\cfrac{\partial}{\partial w_j}J(\vec{w},b) = \cfrac{\partial J}{\partial L}\cfrac{\partial L}{\partial \hat{y}}\cfrac{\partial \hat{y}}{\partial z}\cfrac{\partial z}{\partial w_j}$$

From the right,
$$\begin{aligned}
&z=\vec{w}\cdot\vec{x}+b=w_1x_1+w_2x_2+\cdots+w_jx_j+\cdots+w_mx_m+b\\
&\rightsquigarrow\cfrac{\partial z}{\partial w_j}=x_j&(1)\\
\\
&\hat{y}=\cfrac{1}{1+e^{-z}}\\
&\rightsquigarrow\cfrac{\partial \hat{y}}{\partial z}=\cfrac{e^{-z}}{\left(1+e^{-z}\right)^2}=\cfrac{1}{1+e^{-z}}\left(\cfrac{1+e^{-z}}{1+e^{-z}}-\cfrac{1}{1+e^{-z}}\right)=\hat{y}\left(1-\hat{y}\right)&(2)\\
\\
&L=-\left\{y\log{\hat{y}+(1-y)\log{(1-\hat{y})}}\right\}\\
&\rightsquigarrow\cfrac{\partial L}{\partial \hat{y}}=-\left\{\cfrac{y}{\hat{y}}+\cfrac{-(1-y)}{1-\hat{y}}\right\}=\cfrac{\hat{y}-y}{\hat{y}\left(1-\hat{y}\right)}&(3)\\
\\
&J=\cfrac{1}{m}\sum^m_{i=1}L^{(i)}\\
&\rightsquigarrow\cfrac{\partial J}{\partial L^{(j)}}=\cfrac{1}{m}&(4)\\
\end{aligned}$$
Hence,

$$\cfrac{\partial}{\partial w_j}J(\vec{w},b) = \cfrac{1}{m}\cfrac{\hat{y}-y}{\hat{y}\left(1-\hat{y}\right)}\hat{y}\left(1-\hat{y}\right)x_j = \cfrac{1}{m}\left(\hat{y}-y\right)x_j$$

Additionally,
$$\cfrac{\partial z}{\partial b}=1\implies\cfrac{\partial}{\partial b}J(\vec{w},b) =\cfrac{1}{m}\left(\hat{y}-y\right)$$

Consequently,
$$
\begin{aligned}
w - \alpha \cfrac{\partial}{\partial w} J(w,b)=w-\alpha\cfrac{1}{m}\sum^m_{i=1}\left(f_{\vec{w},b}(\vec{x}^{(i)})_-y^{(i)}\right)\vec{x}^{(i)}& \quad \overset{\text{sub}}{\rightsquigarrow} \quad w \\
b - \alpha \cfrac{\partial}{\partial b} J(w,b)=b-\alpha\cfrac{1}{m}\sum^m_{i=1}\left(f_{\vec{w},b}(\vec{x}^{(i)})_-y^{(i)}\right)& \quad \overset{\text{sub}}{\rightsquigarrow} \quad b
\end{aligned}
$$

Thus, the expression is the same as in a regression model. (Note that the form of the model $f$ may differ.)
