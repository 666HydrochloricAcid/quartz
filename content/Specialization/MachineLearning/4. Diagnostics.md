# 4. 1 Overfitting 
---
Overfitting refers to a situation where a model performs well on the training data but fails to generalize to unseen data because it has focused too much on the training set. Visually, an overfitted model would appear too wiggly or distorted to pass through every data point instead of fitting the general trend.

## 4. 1. 1 Training Set / Validation Set / Test set
---
Our first step is to divide the dataset into three splits. 60~80% of the data is used as the **training set**, which is used to train the model. During this process, you'll likely want to adjust the hyperparameters multiple times to improve the accuracy. At this point, you adjust the hyperparameters in a way that minimizes the cost on the **validation set**. **Test set** is only used for final evaluation not modifying the model.
