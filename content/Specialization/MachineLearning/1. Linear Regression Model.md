# 1. 0 Introduction to Regression
---
A linear regression model (or linear model for regression) is a model that predicts an output for a sample represented by a **continuous number** or a vector of continuous numbers.
Let's dive into the Housing Problem

### § Housing Problem
![feature 1](ML1_1.png)

Problem Statement : There are five data points, each containing the square footage and the price of a house. Given the square footage of a new house, can we predict its price?
### Data Table
| $x$ | $y$ |
| :-: | :-: |
|  1  | 300 |
| 1.2 | 375 |
| 1.6 | 450 |
| 1.8 | 475 |
| 2.0 | 500 |

### Terminology
- training set : data set used to train the model
- $x$ : **feature**, input variable (size)
- $y$ : **target**, output variable (price)
- $m$ : number of training examples ($m=5$)
- $(x^{(i)}, y^{(i)})$ : $i$ th training example ($(x^{(1)}, y^{(1)}) = (1, 300)$)

### Model
In this problem, the model takes the size as input and outputs the predicted price, say $\hat{y}$.
We represent model $f$ as follow:
$$f_{w,b}(x)=wx+b(=\hat{y})$$
With proper parameters $w$ and $b$, we can draw the blue line as below.
![feature 2](ML1_2.png)
# 1. 1 Cost Function
---
Eventually, our goal is to find the best parameters $w$ and $b$ such that the prediction of $i$ th sample $\hat{y}^{(i)}$ is close to the target value $y^{(i)}$ for all samples. Hence, we quantify the errors between predictions and targets, which is referred to as the cost function defined as follow:
$$\begin{align}J(w,b)=\cfrac{1}{m}\sum^{m}_{i=1}\left(\hat{y}^{(i)}-y^{(i)}\right)^2\\
=\cfrac{1}{m}\sum^{m}_{i=1}\left(f_{w,b}\left(x^{(i)}\right)-y^{(i)}\right)^2\end{align}$$
where $\left(f_{w,b}\left(x^{(i)}\right)-y^{(i)}\right)$ is called error term. This cost function is called the **Mean Squared Error** function or **MSE Loss**.
Our goal is to find $w$ and $b$ that minimize the value of $J$ : $min_{w,b}J(w,b)$
# 1.2 Gradient Descent
---
If we plot $J(w,b)$ with respect to $w$ and $b$, we obtain the graph as below.
_I plotted a red point with random values of $w$ and $b$ as our initial parameters._
![feature 3](ML1_3.png)
Shifting the red point to a lower position on the surface corresponds to minimizing the value of cost function. With partial derivatives of $J(w,b$) with respect to $w$ and $b$, let's determine the direction and magnitude of the step the red point should take.

1. Set an appropriately small number, $\alpha=0.001$, say **learning rate**.

2. Update parameters as follow:
$$\begin{aligned}w-\alpha\cfrac{\partial}{\partial w}J(w,b)\overset{\underset{\mathrm{sub}}{}}{\rightsquigarrow}w\\
b-\alpha\cfrac{\partial}{\partial b}J(w,b)\overset{\underset{\mathrm{sub}}{}}{\rightsquigarrow}b\end{aligned}$$
3. Repeat step 2 until the red point arrives near the minimum.
4. We now know how to set $w$ and $b$.
Improper value of learning rate can cause...
- slow updates when $\alpha$ is too small or
- overshoot the minimum when $\alpha$ is too large
Slow updates may require a higher number of iterations to converge to minimun, which can increase the computational cost.

On the other hand, overshoots can make the point not to converge but either diverge or oscillate.
