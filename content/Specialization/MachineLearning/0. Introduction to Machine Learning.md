# What is ML?
---
ML is a field of study that focuses on enabling computers to learn from data and improve their performances on specific tasks without being explicitly programmed.
# Course Overview
---
### 1. Supervised Learning
- Supervised learning is a field of ML where a model learns from input data and **output labels**.
- This includes **Regression** and, **Classification**.
### 2. Advanced Learning Algorithm
- In this section, we will cover the overall process of building an ML system and advanced algorithms.
- This includes **Neural Network**, **Diagnostics**, and **Decision Tree**.
### 3. Unsupervised Learning, Recommender System, Reinforcement Learning
- Unsupervised learning, in contrast to supervised learning, learns from input data **without output labels**.
- A recommender system is a practical tool that analyzes **user** preferences and recommends insightful **items**.
- Reinforcement learning is a field of machine learning where an agent learns **optimal actions** through trial and error by receiving **rewards** or penalties from its environment.
